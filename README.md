# Hi there, I'm Amal Babu 👋
**Full-Stack Software Engineer · AI Integration & Agentic Engineering**

I build full-stack **ERP & CRM systems** with domain-specific workflows across **Healthcare**, **Construction**, and **E-commerce** — from requirements gathering to system architecture and end-to-end delivery. I **lead the technical team from the ground up**, setting engineering direction, establishing coding standards, and driving consistent, high-quality output across projects.

On the AI side, I develop **AI agents** and intelligent automation using **Pydantic AI** and **LangGraph**, embedded directly into business software. I also practice **agentic engineering** as a coding discipline — using **GitHub Copilot**, **Claude Code**, and **OpenAI Codex** with deliberate engineering judgment to ship faster without compromising quality.

I care deeply about **software craft** — clean architecture, separation of concerns, maintainable code, and systems designed to grow without becoming fragile. I authored the **[Engineering Standards @ Hybrid Interactive](https://engineering.hybridinteractive.in/)** — a living documentation hub covering architecture patterns, tech stack decisions, frontend/backend standards, and AI-assisted coding workflows used across the team.
